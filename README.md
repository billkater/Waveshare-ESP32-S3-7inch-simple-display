# Waveshare-ESP32-S3-7inch-simple-display
This is a simple YAML for a Waveshare display. It makes 2 columns of 5 rows. Controls all my garage doors and walk through doors. The icons change color and style depending on state of doors and locks. Locks even flash yellow when batteries are low.
